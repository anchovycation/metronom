constructor

  • new default(schema: Schema, keyPrefix?: string, modelOption?: ModelOptions): default

  • Represents a Metronom ORM Model

    Returns

    new record of Model

    Parameters

    • schema: Schema

      Record's key-value schema

    • keyPrefix: string = 'object'

      Record unique key's prefix. "users:1234" --> "keyPrefix:keyUnique"

    • Optional modelOption: ModelOptions

      Optional model settings. It's include 3 key.

      • keyUnique: it's unique part of model key
      • flexSchema: Normally, you can't define any key except the fields in schema, but if this value is true, you can only add a value to the schema by giving it keyUnique
      • redisClientOptions: node-redis client options.

    Returns default
